# p6: Factorial Challenge

## 題目描述
計算給定數字的階乘,必須使用遞迴實作。

## 輸入格式
一個非負整數 n (0 ≤ n ≤ 20)

## 輸出格式
n 的階乘值

## 範例

### 輸入 1
```
5
```

### 輸出 1
```
120
```

### 輸入 2
```
0
```

### 輸出 2
```
1
```

### 輸入 3
```
7
```

### 輸出 3
```
5040
```

## 配分
- 總分: 25 分
- 測資數量: 5 個

## 限制條件
- **禁止使用**: `for`, `while`, `do`, `goto`
- **必須包含**: `factorial` 函數
- **超時限制**: 1 秒

## 注意事項
✓ 此題為正確的遞迴實作範例,所有測資都會通過
