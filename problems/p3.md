# p3: String Manipulation

## 題目描述
計算輸入中有多少個單字(以空白分隔)。

## 輸入格式
多個單字,以空白分隔

## 輸出格式
一個整數,表示單字的數量

## 範例

### 輸入 1
```
hello world test case
```

### 輸出 1
```
4
```

### 輸入 2
```
a
```

### 輸出 2
```
1
```

- 測資數量: 5 個
