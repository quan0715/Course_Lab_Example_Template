# p5: Factorial (Recursion)

## 題目描述
計算給定數字的階乘,必須使用遞迴實作。

## 輸入格式
一個非負整數 n (0 ≤ n ≤ 20)

## 輸出格式
n 的階乘值

## 範例

### 輸入 1
```
5
```

### 輸出 1
```
120
```

### 輸入 2
```
0
```

### 輸出 2
```
1
```

### 輸入 3
```
10
```

### 輸出 3
```
3628800
```

## 配分
- 總分: 25 分
- 測資數量: 5 個

## 限制條件
- **禁止使用**: `for`, `while`, `do`, `goto`
- **必須包含**: `factorial` 函數
- **超時限制**: 1 秒

## 注意事項
✗ 此題目前使用 `for` 迴圈實作,會違反關鍵字限制 (用於展示關鍵字違規)
