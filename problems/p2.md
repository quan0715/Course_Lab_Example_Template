# p2: Array Operations

## 題目描述
反轉每一行輸入的字串。

## 輸入格式
多行字串

## 輸出格式
每行字串反轉後的結果

## 範例

### 輸入 1
```
abc
```

### 輸出 1
```
cba
```

### 輸入 2
```
hello
world
```

### 輸出 2
```
olleh
dlrow
```

## 配分
- 總分: 25 分
- 測資數量: 5 個
- 特殊配分:
  - 測資 01: 10 分
  - 測資 02: 10 分
  - 測資 03: 5 分
