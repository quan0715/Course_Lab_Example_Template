# 操作說明

## 📖 如何使用測試系統

### 1. 執行測試
- **執行所有測試**：點擊右上角的「執行所有測試」按鈕
- **執行單一題目**：點擊題目卡片開啟詳情，進入「程式碼」分頁後點擊「執行測試」


### 2. 查看結果
- **總覽**：主頁顯示每個題目的通過/失敗狀態和得分
- **詳細資訊**：點擊題目卡片查看詳細的測試結果
- **測試案例**：展開每個測試案例查看輸入、預期輸出和實際輸出

### 3. 狀態說明
- **AC (Accepted)**：所有測試通過 ✅
- **WA (Wrong Answer)**：答案錯誤 ❌
- **TLE (Time Limit Exceeded)**：執行逾時 ⏱️
- **CE (Compilation Error)**：編譯錯誤 🔧
- **KW (Keyword Check)**：關鍵字檢查失敗 🔍
- **ERR (Runtime Error)**：執行錯誤 ⚠️

### 4. 題目資訊
- **配分**：該題目的總分
- **Timeout**：執行時間限制
- **Forbidden**：禁止使用的關鍵字（紅色標籤）
- **Required**：必須使用的關鍵字（綠色標籤）

### 5. 評分規則
只有當題目的**所有測試案例都通過**時，才會獲得該題目的分數。

## 🛠️ 如何新增題目

本系統提供互動式腳本協助您快速新增題目。

### 使用方式
在終端機執行以下指令：
```bash
python3 add_problem.py
```

### 腳本功能
1. **建立檔案結構**：自動建立測資目錄、題目說明檔與程式模板
2. **更新配置**：自動更新系統設定檔
3. **引導式輸入**：只需回答幾個簡單問題（題目名稱、配分等）

### 完成後
腳本執行完畢後，您只需要：
1. 編輯 `problems/pX.md` 填寫題目描述
2. 編輯 `src/pX.cpp` 實作解答
3. 在 `tests/pX/` 目錄下填入測資
